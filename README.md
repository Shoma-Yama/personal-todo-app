# Personal ToDo App

自分専用の ToDo アプリを開発するためのプロジェクトです。

## Current Status

この環境では `Node.js` / `npm` が未導入のため、実装前の仕様整理から進めています。

## Planned Stack

- Frontend: Next.js
- Styling: Tailwind CSS
- Backend / DB / Auth: Supabase
- Deploy: Vercel
- Mobile support: Responsive Web App
- Future enhancement: PWA

## Product Direction

- PC とスマホの両方で使う
- 急な頼み事を瞬時に追加できる
- 全タスクを一目で俯瞰できる
- 期限、優先度、カテゴリ、タグで整理できる
- 近未来的で洗練されたデザインを目指す

詳細な仕様は `docs/spec.md` を参照してください。
